# **Olá 👋 Eu sou Delmar de Lima**

==============================

### **Contador de Profissão & Desenvolvedor**

Com mais de 6 anos de experiência desenvolvendo com **PHP** e **Delphi**, meu maior destaque é o desenvolvimento de sistemas de gestão avançados. Possuo forte conhecimento em Linux, Cloudflare R2 (compatível com AWS S3) e arquitetura baseada em Microsserviços.

Atualmente, estou aplicando minha visão unificada de Contabilidade e Tecnologia nas seguintes frentes:

* 🏢 **Amil Contabilidade:** Desenvolvendo um CRM próprio e exclusivo para o escritório.  
* ⚙️ **Automação:** Criando scripts e automações para otimizar rotinas contábeis.  
* 🛒 **Sistemas de Gestão:** Desenvolvendo um **Mini ERP** completo (Catálogo, PDV e emissão de Nota Fiscal) voltado para lojas.

**Sobre mim:**

* 🌍 Eu nasci e moro em Apuí/AM 🇧🇷  
* ✉️ Você pode contatar-me via [delmar@amil.cnt.br](mailto:delmar@amil.cnt.br)  
* 🧠 Estou sempre estudando e aprimorando minha stack (atualmente focado em ecossistema Python e React Native).
